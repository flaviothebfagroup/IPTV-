# BFA Link Page (no wordmark)

## Files
- `index.html` = public page
- `links.json` = your data (links, titles, etc.)
- `editor.html` = no-code editor (runs in your browser)
- `styles.css`, `app.js`, `editor.js` = logic + styles
- `assets/` = your images

## Quick use (GitHub Pages)
1. Upload everything to a GitHub repo
2. Settings → Pages → Deploy from branch → `main` (root)
3. Open:
   - `https://YOURUSER.github.io/YOURREPO/` (main page)
   - `https://YOURUSER.github.io/YOURREPO/editor.html` (no-code editor)

## Updating links without code
1. Open `/editor.html`
2. Edit your links
3. Click **Download links.json**
4. Replace `links.json` in GitHub (upload or edit file)
